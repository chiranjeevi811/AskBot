# 🤖 AskBot

An AI-powered chatbot built with the **MERN Stack** and **Groq API** that enables users to have real-time conversations with an intelligent assistant. The application supports persistent chat history, multiple conversation threads, and a modern ChatGPT-inspired interface.

## 🚀 Live Demo

### 🌐 Frontend
https://ask-bot-jrht.vercel.app/

### ⚙️ Backend API
https://askbot-0htc.onrender.com

---

## ✨ Features

- 💬 Real-time AI conversations
- 🧠 Powered by Groq LLM API
- 📝 Persistent chat history using MongoDB
- 📂 Multiple conversation threads
- ➕ Start new conversations
- 🗑️ Delete previous chats
- 📜 Sidebar with conversation history
- ⚡ Fast and responsive React UI
- 🌐 Fully deployed on Vercel & Render

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/chiranjeevi811/AskBot.git
```

### Navigate into the project

```bash
cd AskBot
```

---

## Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file inside the **backend** folder.

```env
PORT=8080
MONGODB_URI=YOUR_MONGODB_URI
GROQ_API_KEY=YOUR_GROQ_API_KEY
```

Run the backend

```bash
npm start
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
