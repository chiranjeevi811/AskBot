# 🤖 AskBot - AI Chatbot Backend

An AI-powered chatbot backend built with **Node.js, Express, MongoDB, and Groq AI**.  
AskBot provides intelligent conversational responses through an API-powered architecture.

🚀 **Live Backend:** https://askbot-0htc.onrender.com

---

## ✨ Features

- 🤖 AI-powered conversational responses
- ⚡ Fast API-based chatbot interaction
- 🔐 Secure API key management using environment variables
- 🌐 RESTful API architecture
- 🗄️ MongoDB database integration
- ☁️ Deployed on Render
- 🔄 CORS enabled for frontend integration

---

## 🛠️ Tech Stack

### Backend
- **Node.js**
- **Express.js**
- **MongoDB**
- **Mongoose**
- **Groq SDK**
- **OpenAI SDK**

### Tools & Deployment
- Git & GitHub
- Render
- MongoDB Atlas
- dotenv
